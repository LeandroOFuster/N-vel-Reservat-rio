# Controle de Níveis de Água

Sistema simples de monitoramento de reservatório que exibe alertas coloridos no terminal.

## Como usar

Instale a dependência:
```
pip install colorama
```

Execute o programa:
```
python controle_nivel_agua.py
```

## Níveis

| Nível | Situação | Cor |
|-------|----------|-----|
| 1 | Muito baixo (crítico) | Vermelho |
| 2 | Baixo | Amarelo |
| 3 | Médio | Verde |
| 4 | Alto | Ciano |
| 5 | Muito alto (alerta) | Azul |
